# Day 1 사칙연산
## 두 수의 합
```js
function solution(num1, num2) {
    return num1 + num2;
}
```

## 두 수의 차
```js
function solution(num1, num2) {
    return num1 - num2;
}
```

## 두 수의 곱
```js
function solution(num1, num2) {
    return num1 * num2;
}
```

## 몫 구하기
```js
function solution(num1, num2) {
    return Math.floor(num1 / num2);
}
```
